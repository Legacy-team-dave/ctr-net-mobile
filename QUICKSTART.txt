=========================================================
  ENROL.NET Mobile — Guide de démarrage rapide
=========================================================

PRÉREQUIS
---------
- Node.js 22+
- npm 10+
- Java JDK 21 (Temurin)
- Android SDK API 36, build-tools 35.0.0
- Android Studio (recommandé pour debug)
- Serveur web central opérationnel (`ctr.net-fardc`)

INSTALLATION
------------
1. Cloner le dépôt :
   git clone https://github.com/Legacy-team-dave/ctr-net-mobile.git
   cd ctr-net-enrollement-mobile

2. Installer les dépendances :
   INSTALL.bat
   ou
   npm install

3. Lancer en navigateur (dev) :
   START.bat
   ou
   npm start

   → Ouvrir http://localhost:4200

CONFIGURATION
-------------
Au premier lancement de l'application (navigateur ou APK) :
1. Le splash natif Capacitor s'affiche brièvement, puis le splash Angular `ENROL.NET`
2. La configuration serveur reste accessible tant qu'aucune IP valide n'est enregistrée
3. Saisir manuellement l'adresse IPv4 du PC serveur (ex: 10.71.62.9)
4. Vérifier que le téléphone et le PC sont sur le même réseau Wi-Fi
5. Cliquer sur "Tester la connexion"
6. Si OK → l'IP est sauvegardée et la page login s'affiche

CONNEXION
---------
- Seuls les utilisateurs avec le rôle `ENROLEUR` peuvent se connecter
- Saisir identifiant et mot de passe
- Le token est stocké localement pour les requêtes suivantes

UTILISATION — ENRÔLEMENT
------------------------
1. Onglet `ENROL.NET` (actif par défaut)
2. Capture de la photo du militaire
3. Capture / association des empreintes
4. Scan du QR affiché côté web
5. Vérification des informations personnelles
6. Validation finale du dossier
7. Synchronisation immédiate ou différée

UTILISATION — PROFIL
--------------------
- Onglet `Profil` : affiche les informations du compte connecté
- Onglet `Quitter` : déconnexion avec confirmation

COMPILATION APK
---------------
Méthode recommandée : GitHub Actions (automatique à chaque push)
Artifact GitHub Actions : `enrol-net-enrollement-mobile-apk-v<version>`

Méthode script batch :
   BUILD_APK.bat

Méthode script PowerShell :
   .\build_apk.ps1
   .\build_apk.ps1 -InstallOnDevice

Méthode manuelle :
   npm run build
   npx cap sync android
   cd android
   ./gradlew assembleDebug

L'APK local source se trouve dans :
   android/app/build/outputs/apk/debug/app-debug.apk

L'APK distribuable local est copié dans :
   dist/apk/enrol-net-enrollement-mobile-latest-debug.apk

Installation rapide sur appareil Android connecté :
   INSTALL_APK.bat

Remarque : l'APK local sert au test. L'APK officiel de distribution est celui généré par GitHub Actions.

DÉPANNAGE
---------
- "Connexion impossible" : vérifier que l'IP est correcte et que
  le serveur web tourne (Laragon démarré, Apache actif)
- "Accès refusé" : seul le rôle `ENROLEUR` est autorisé
- "QR non trouvé" : vérifier qu'il provient bien du web et qu'il est lié à un militaire vivant
- Écran blanc : vérifier la console (F12 en navigateur),
  relancer avec npm start
